## 命令
```bash
mvn flyway:migrate
#使用标准 Maven 命令行属性将参数传递给目标。例如：
mvn -Dmybatis.generator.overwrite=true mybatis-generator:generate
#这将运行 MBG 并指示它覆盖它可能找到的任何现有 Java 文件。
```
